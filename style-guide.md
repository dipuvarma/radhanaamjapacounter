# Style Guide for RadhanaamJapaCounter App

## 1. Overall

**Purpose**: Track daily chanting of "Radhe Radhe" (राधे राधे)

**Target Audience**: Devotees practicing Japa

**Tone**: Devotional, peaceful, encouraging

## Typography

### Font Family

- Main Font (Headings, Titles): Cinzel Decorative
- Secondary Font (Body, Stats): Poppins

### Font Size

- For Heading (h1) : sm:text-3xl, md:text-5xl, lg:text-6xl
- For Sub Heading (h2) : sm:text-xl, md:text-3xl, lg:text-4xl
- For Sub Heading (h3) : sm:text-lg md:text-2xl lg:text-3xl
- For Paragraph (p) : sm:text-sm, md:text-base
- For Buttons (button) : sm:text-sm, md:text-base

## Color Palette (Saffron and Yellow)

1. #F37420 (Orange)
2. #F9BB4D (Yellow),
3. #FAE6B8 (Light Yellow),
4. #FAD199 (Saffron)
